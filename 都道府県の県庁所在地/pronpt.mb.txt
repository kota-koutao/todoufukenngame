ゲーム仕様書：県庁所在地クイズシューティング（47都道府県対応版）
1. ゲーム概要
47都道府県が敵としてランダムに出現する。プレイヤーは、表示される3つの選択肢から正しい県庁所在地を選んで敵を撃ち落とすシューティングゲーム。

2. ゲーム画面の要素
プレイヤー：画面下部を左右に動かせる自機。

敵：都道府県の名前が書かれたキャラクター。画面上部から出現し、まっすぐ下に移動する。

選択肢ボタン：画面下部に3つのボタンを配置。

スコア表示：画面の隅に現在のスコアを表示。

ライフ表示：画面の隅に残りライフ（❤️ x 3など）を表示。

3. ゲームデータ
ゲームで利用する47都道府県と県庁所在地のペアリストを用意する。

データ形式: JavaScriptの配列として定義する。

JavaScript

const quizData = [
  { prefecture: '北海道', capital: '札幌市' },
  { prefecture: '青森県', capital: '青森市' },
  { prefecture: '岩手県', capital: '盛岡市' },
  // ... 以下、47都道府県分をすべて定義 ...
  { prefecture: '沖縄県', capital: '那覇市' }
];
4. ゲームの流れ
スタート：ライフ3、スコア0でゲーム開始。

問題の選定：

quizDataの47個のデータの中から、まだ出題されていないものをランダムに1つ選ぶ。これを「正解のペア」とする。

47都道府県すべてを一度出題したら、出題履歴をリセットする（2周目に入る）。

敵の出現：選ばれた「正解のペア」のprefecture（例：「岩手県」）が敵として画面上部に出現する。

選択肢の表示：

正解の選択肢：「正解のペア」のcapital（例：「盛岡市」）を用意する。

ダミーの選択肢：quizData全体の中から、正解以外のcapitalをランダムに2つ選ぶ（例：「新潟市」「水戸市」）。

表示：上記3つの選択肢（正解1つ、ダミー2つ）の順番をシャッフルし、3つのボタンにそれぞれ表示する。

プレイヤーの操作：正解だと思うボタンをクリックする。

判定：

正解の場合：弾が発射されて敵を破壊。「ピンポン！」音を鳴らし、スコアを10点加算する。その後、ステップ2に戻る。

不正解の場合：「ブブー」音を鳴らす。弾は発射されず、選択肢ボタンが1秒間押せなくなる。

敵を見逃した場合：敵が画面下に到達したら、ライフを1つ減らす。敵は消滅し、次の問題（ステップ2）に進む。

ゲームオーバー：ライフが0になったら「ゲームオーバー」と表示してゲームを停止する。
